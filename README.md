# Power BI Workshop Demo

![Power BI Dashboard](Screenshot%202025-12-25%20210606.png)

## Overview
This project is a demo created based on a **Power BI Workshop** that I attended.  
The workshop focused on building interactive dashboards, analyzing data effectively, and applying best practices in data visualization to support data-driven decision-making.

## Workshop Highlights
- Introduction to Power BI     
- Designing interactive dashboards  
- Data visualization best practices  

## Project Description
The dashboard analyzes **sales performance** across different products, cities, and managers.  
It provides a clear overview of key performance indicators (KPIs) and enables users to explore the data using interactive filters.

## Key Features
- KPIs: Total Sales, Profit, Units Sold  
- Sales trends over time  
- Sales analysis by product and city  
- Interactive slicers for deeper insights  

## Tools Used
- Microsoft Power BI  
- CSV Dataset  
- DAX for calculations  

## Notes
This project was created for learning and demonstration purposes as part of the Power BI workshop.
